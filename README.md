#  Introductory Course to Machine Learning  
##  Topic 1: The Library
This first assignment consist on developping a console application that allows managing books in a library. 
The system includes the following features, accessible through an interactive menu in the terminal:
### Features
1.  Add book  
2.  Search by title  
3.  Search by author  
4.  Borrow book  
5.  Return book  
6.  List all books  
7.  Exit
